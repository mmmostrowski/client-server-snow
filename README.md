## Quick run

1. Run in terminal:
```bash
docker compose up 
```
2. Visit http://localhost:3000/

## Develop

1. Run in terminal:
```bash
./dev docker compose up --build
```
2. Visit http://localhost:8181/


## Develop in IDE

1. Start `bootRun` gradle action in IDE
2. Run in terminal:
```bash
./dev docker compose up reactjs-client --build 
```

Visit http://localhost:8181/
